# Test Pyramid

## Layer Allocation

- Unit tests: 60%
- Integration tests: 25%
- End-to-end tests: 10%
- Load and resilience tests: 5%

## Unit Test Focus

- JSON schema validation
- Safety policy evaluation
- Path canonicalization
- Metadata repository operations
- Tool result envelope generation

## Integration Test Focus

- MCP tool dispatch to controller bridge
- Blender launch and attach flow
- Snapshot creation and rollback metadata
- Export gating and QA generation

## End-to-End Focus

- Create project to preview render
- Local revision loop
- Export workflow with QA gate
