# End-to-End Testing

## Core E2E Scenarios

1. Create a small SF drone, materialize it, preview it, and export it.
2. Create a premium furniture asset, revise one part locally, re-render, and compare snapshots.
3. Create a building exterior shell, inspect scale and naming, fix issues, and export.
4. Assemble a small display scene around an existing asset and render multiple views.

## Success Criteria

- Each scenario completes without manual Blender scripting.
- Structured history records exist for each step.
- Preview image paths, snapshot identifiers, and export outputs are all available.
