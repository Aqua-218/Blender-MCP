# Recommended Team Structure

## Minimal Team

- 1 platform engineer for MCP server and metadata layer
- 1 Blender pipeline engineer for controller and domain modules
- 1 QA-minded technical artist for asset validation, presets, and acceptance testing

## Expanded Team

- Add 1 infrastructure engineer when hosted HTTP mode is introduced
- Add 1 product-facing technical designer for prompt patterns, tool descriptions, and UX refinement

## Ownership Boundaries

- MCP platform engineer owns schemas, policy, history, transport, and packaging
- Blender pipeline engineer owns bpy execution modules and export behavior
- Technical artist owns presets, quality criteria, and workflow validation
