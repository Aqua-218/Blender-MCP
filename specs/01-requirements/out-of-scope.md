# Out of Scope

The following capabilities are intentionally excluded from the initial production boundary:

- Commercial-grade fully automatic humanoid character creation
- Guaranteed production-ready auto-rigging for complex organic characters
- Guaranteed AAA-level game optimization across all target engines
- Direct control of non-Blender DCC applications
- General-purpose workstation or operating-system automation
- Arbitrary shell, package-manager, or network execution initiated by the LLM through MCP tools
- File access outside operator-approved workspace roots
- Deliberate recreation of copyrighted characters, trademarked products, or restricted IP likenesses
- Detailed real-world weapon or dangerous-device generation intended for practical replication
- Multi-user concurrent editing of the same Blender runtime
- Full cloud render-farm orchestration in the MVP

The following capabilities remain future candidates but are not committed for the MVP:

- Autonomous multi-variant concept exploration with ranking
- Fully procedural city-scale worlds with strong streaming guarantees
- Advanced animation libraries with motion retargeting
- Deep asset-library publishing workflows across many blend files
