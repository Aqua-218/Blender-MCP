# User Story Map

## Epics

- [epic-01-project-lifecycle.md](epic-01-project-lifecycle.md): Start, save, snapshot, compare, and recover projects
- [epic-02-asset-generation.md](epic-02-asset-generation.md): Create and revise individual assets with semantic part structure
- [epic-03-scene-world-pipeline.md](epic-03-scene-world-pipeline.md): Compose scenes and worlds from assets and procedural systems
- [epic-04-quality-export-iteration.md](epic-04-quality-export-iteration.md): Inspect, repair, optimize, export, and track revisions

## Story Priorities

- Must-have: MVP stories required for project, model, preview, QA, save, and export
- Should-have: Scene composition, scattering, world hierarchy, multi-view render, richer QA automation
- Could-have: Experimental rigging, advanced animation, and expansive world editing workflows
